# Teste Técnico - Lista de Tarefas

## Tecnologias utilizadas

* Angular
* .NET
* PostgreSQL

## Como rodar o projeto

### Backend

```bash
cd backend
dotnet run
```

### Frontend

```bash
cd frontend
npm install
ng serve
```

## Funcionalidades

* Criar tarefas
* Listar tarefas
* Deletar tarefas

## Observações

* Comunicação via API REST
* Atualização automática da lista com RxJS (Subject)
